# Adder

Simple calculator program

Uses ReactJS, WebPack, Babel, Jest.
Due to a limitation with Jest & Jsdom, tests need to be run in node 0.10. 

## Installation

```zsh
npm install
npm run build
npm start
```

## Test

```zsh
npm test
```

## Development

```zsh
npm run watch
```
